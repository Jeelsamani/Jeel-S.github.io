Methodology

How It Works

The core formula V = (N × W × ACV) / C is decomposed using partial derivatives — isolating the marginal contribution of each variable to total velocity change.

Each component is then segmented by rep archetype (Challenger, Relationship Builder, Consultant, Lone Wolf), deal size band, and industry vertical to find which exact combination is underperforming.

The result: instead of "win rate dropped 3%," the system tells you exactly which rep type, selling which deal size, in which stage, lost which dollar amount.

Dataset
847 synthetic SaaS deals across 4 segments, 5 verticals, and 12 reps — generated with realistic distributions from industry benchmarks.
Decomposition Method
Variance decomposition using Shapley values — the same method used in game theory to fairly attribute group outcomes to individual contributors.
Output
A ranked micro-failure table showing which velocity component caused which dollar loss — ready to hand to a VP of Sales for immediate action.
